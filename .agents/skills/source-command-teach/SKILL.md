# source-command-teach

Use this skill when the user asks to run the migrated source command `teach`.

## Command Template

罗迦南是量化新手。识别最近实验中涉及的核心量化概念，用通俗易懂的方式解释。

输出格式：
1. **概念名称**：一句话定义
2. **为什么需要它**：在比赛中它解决了什么问题
3. **直观理解**：用类比或简单例子说明（避免纯数学公式）
4. **在我们的策略中怎么用**：实际应用场景
5. **延伸阅读**：如果罗迦南想深入学习，推荐的下一步方向

将这些内容追加写入 `量化学习/` 目录，以主题命名（如 `03-夏普比率.md`）。
