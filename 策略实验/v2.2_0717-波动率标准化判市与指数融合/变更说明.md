# v2.2 变更说明 (2026-07-17)

## 从 v2.1 到 v2.2 的核心变化

### Phase 1: 判市阈值波动率标准化

**文件**: `quant/market_regime.py`

**问题**: `detect()` 用绝对阈值 3% 判断牛熊，在高波市场（日波 3%）3% = 1σ 噪声，低波市场（日波 0.5%）3% = 6σ 永不到达。

**修改**: 新增 5 个类常量（BULL_PRICE_SIGMA=2.0, BULL_SLOPE_SIGMA=0.5 等），用 20 日市场波动率标准化阈值：
- `price_sigma = price_vs_ma / vol_daily`
- `slope_sigma = ma_slope / vol_daily`
- 地板: `max(0.01, vol_daily * 1.0)`

**效果**: 同一阈值在各波动环境下语义一致——"偏离 MA60 超过 2 个日标准差"在任何市场都代表同样强度的信号。

### Phase 2: CSI 300 指数独立判市 + 融合

**新增文件**:
- `quant/market_index.py`: 基于 CSI 300 指数的判市（用同一套波动率标准化逻辑）
- `quant/regime_fusion.py`: 双信号加权投票融合

**融合逻辑**: S_tech(0.5) + S_index(0.3) → 一致时输出共识，分歧时偏保守（range）

**评分**: 78.5/100 (持平 v2.1，因为两个信号在当前数据上高度一致)

### 附加修复

- `scripts/run_quant_pipeline.py`: 补充缺失的 `MarketRegime` 导入
- `evaluation/scoring.py`: 改用 `RegimeFusion.detect()` 替代 `MarketRegime.detect()`
