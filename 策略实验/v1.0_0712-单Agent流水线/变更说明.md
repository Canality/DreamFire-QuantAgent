# v1.0 变更说明 (2026-07-12)

## 初始版本

### 架构
- 单 Agent 流水线: 数据获取 → 因子计算 → 选股 → 仓位分配 → 回测 → 报告
- 基于 JiuwenSwarm 框架的基础集成
- 5 因子模型 (动量20/60 + 波动率 + 最大回撤 + 成交量趋势)

### 策略特点
- 行业中性化 Z-score 标准化
- 风险平价仓位管理 (逆波动率加权)
- 每板块至少 1 只, 最多 15 只持仓

### 评分
- 评分框架尚未建立 (scoring.py 在 v2.0 才创建)
- 回测指标: 累计收益 +18.87%, 最大回撤 4.77%, Sharpe 5.46

### 数据源
- yfinance 单源, 49只股票

### 局限性
- 单 Agent 无多空辩论
- 硬编码阈值
- 无多源数据兜底
