# 策略实验目录

每个子目录对应一次版本迭代，包含该版本的完整产物和变更说明。

## 目录命名规范

```
v{major}.{minor}_{日期MMDD}-{简短描述}/
```

## 每个版本必须包含

| 文件 | 内容 |
|------|------|
| `评分.json` | 自评估得分详情 |
| `投资组合.json` | 最终选股与仓位 |
| `投资报告.md` | 完整量化分析报告 |
| `变更说明.md` | 本版本相对上一版本的改动 |
| `资源消耗.md` | Token/运行时/CPU 统计 (v2.1+) |
