# v2.1 变更说明 (2026-07-17)

## 从 v2.0 到 v2.1 的核心变化

### Bug 修复

1. **数据源依赖缺失**: venv 中未安装 yfinance/akshare → 通过阿里云镜像安装
2. **数据源顺序**: yfinance 优先 (50s/3只) → akshare 优先 (0.7s/3只)
3. **数据源级联**: 新增 baostock 作为 Tier 2 数据源 (自有服务器，49/49 在 15s 内完成)
4. **工具权限**: 移除 `_build_quant_tools()` 的 leader-only 限制，Bull/Bear 现在可调用 8 个量化工具

### 架构改进

1. **内存缓存**: 添加 `_data_cache` (线程安全)，避免全量价格矩阵通过 LLM 上下文
2. **摘要返回**: `fetch_data()` 返回紧凑摘要而非原始数据
3. **多源兜底**: akshare → baostock → yfinance 三层级联，每层只补充缺失
