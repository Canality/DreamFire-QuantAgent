# 07 — 市场判市与因子权重

## 为什么要判断市场状态

同一个因子在不同市场里，有效性完全不同。

- **牛市**里，动量因子最强——涨的股票继续涨（趋势跟踪）
- **熊市**里，低波动因子最强——抗跌的股票亏得少（风险规避）
- **震荡市**里，反转因子最强——涨多了就跌，跌多了就涨（均值回归）

如果不判断市态、所有市场用同一套因子权重，等于"晴天雨天都穿同一件衣服"。

## 当前判市：两条逻辑 + 一个融合

### 技术面判市 (Phase 1)

看股票池的等权均价：

```
market = 所有股票每天价格的平均
MA60   = 60日均线  (长期趋势参考)
MA20   = 20日均线  (短期趋势参考)

price_vs_ma = (当前价 - MA60) / MA60    ← 价格偏离长期趋势多少
ma_slope    = (MA20 - MA60) / MA60       ← 短期是否在向上

vol_daily   = 最近20天日收益的标准差       ← 用于标准化

price_sigma = price_vs_ma / vol_daily    ← "偏离了几个标准差"
slope_sigma = ma_slope / vol_daily

bull:  price_sigma > 2.0  AND  slope_sigma > 0.5
bear:  price_sigma < -2.0 AND  slope_sigma < -0.5
其他:  range
```

### 为什么要标准化（核心改进）

改之前的代码用的是**绝对阈值**：

```python
# 旧代码
if price_vs_ma > 0.03:  # 偏离超过 3%
    return "bull"
```

问题：3% 在不同波动环境下含义完全不同。

| 市场环境 | 日波动率 | 3% 等于几个标准差 |
|------|------|------|
| 低波动 | 0.5%/天 | 6σ — 几乎永远不会触发 |
| 正常 | 1.5%/天 | 2σ — 合理 |
| 高波动 | 3.0%/天 | 1σ — 噪声，每天随机触发 |

标准化后："偏离超过 2 个标准差"在任何市场里含义一致。

### 指数面判市 (Phase 2)

用沪深 300 指数的实际数据跑同一套逻辑。

为什么不用个股均价？49 只等权平均里，小盘股的噪声比大盘股大得多。沪深 300 是市值加权的，更能代表"真正的市场"。

### 信号融合

```
S_tech  ← 个股均价判市
S_index ← CSI 300 指数判市

如果 S_tech == S_index:  输出共识
如果一方说bull一方说range:  偏保守，看权重
如果一方说bull一方说bear:   不可能，输出range保底
```

### 因子权重随市态调整

不同市态下，同一套因子的倍率完全不同：

| 因子 | 牛市 | 熊市 | 震荡 |
|------|------|------|------|
| 动量_20 | ×1.5 追涨 | ×0.3 不追 | ×0.8 |
| 反转_5 | ×0.3 不抄底 | ×1.5 找超跌 | ×1.3 |
| 波动率惩罚 | ×0.7 放松 | ×1.5 严格 | ×1.1 |
| 回撤惩罚 | ×0.7 放松 | ×2.0 最严 | ×1.0 |

核心理念：**牛市重势（动量），熊市重质（低风险），震荡重反转**。

### 当前判市的最大问题

60 日均线作为趋势判断指标，天生滞后。比如：

```
Window 7: 实际收益 0.78%，波动率 20.72%（其他窗口 8-14%）
判市结果: bull
```

这个窗口明显不是牛市——0.78% 收益配 20% 波动是大震荡特征。但 MA60 还没来得及反映，判市仍然是 bull，导致因子权重错配：动量 ×1.5（继续追涨）+ 反转 ×0.3（忽略超跌反弹机会）。

## 从中学到什么

1. **绝对阈值在不同环境下不可比**：用标准差标准化，让信号含义一致
2. **多源信号融合 > 单一指标**：技术面 + 指数面，共识时才行动，冲突时保守
3. **判市滞后是结构性缺陷**：MA 指标永远滞后于价格，需要用其他手段（如波动率异常检测）来弥补
4. **市态 × 因子权重是关键杠杆**：改一行权重参数的效果可能比加三个新因子还大
