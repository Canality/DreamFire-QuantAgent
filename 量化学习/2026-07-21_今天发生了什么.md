## 一、今天的核心故事（如果你只读一段）

早上我们认为自己的量化策略有 **81.7 分**。到晚上，这个数字被证明是**假的**。

不是因为策略突然变差了。是因为测量工具有三个隐蔽的 bug，系统性地美化了分数——就像用一把刻度偏大的尺子量身高，以为自己 180cm，实际上 170cm。

发现这些问题后，我们用了整整一天修测量工具、重建基线、跑对照实验。最终确定了一个叫 **T2** 的策略候选（得分倾斜配仓），它在诚实的测量下跑出了开发集中位数 **+0.50%**（对比旧生产策略的 **-0.07%**）。

但注意：**这个 +0.50% 不是"新分数"，它只是诚实测量下找到的最佳方向**。我们的策略还没有完成真正的样本外验证。

## 二、今天的核心发现：三个隐蔽的测量 Bug

### Bug 1：前视偏差（Look-Ahead Bias）——"因子偷看了答案"

**问题代码**（`scoring.py` 旧版）：
```python
history = prices_df.iloc[:start_idx + 20]  # ← 包含了未来 20 天！
```

这句话的意思是：算因子时，"历史数据"包含了测试窗口内的行情。这等价于：
- 考试前老师把答案发给了你
- 你用答案去"预测"题目
- 然后宣称自己预测很准

**为什么之前的 81.7 分是假的**：因子在选股时已经知道未来 20 天哪些股票会涨。8/8 窗口正收益、中位数 +4.98%，大部分是"偷看答案"的效果。

**修复**：
```python
history = prices_df.iloc[:start_idx]  # 只看测试窗口之前的数据
```

---

### Bug 2：回测首日收益被排除——"赚了 5% 说没赚"

**问题代码**（`backtest_engine.py` 旧版）：
```python
total_ret = nav[-1] / nav[0] - 1  # ← nav[0] 已经包含了第一天收益！
```

`nav[0]` 不是初始本金，而是 `初始本金 × (1 + 第一天收益率 - 交易成本)`。把它当分母，等于第一天赚的钱被"吃掉了"。

**具体例子**（Codex 复现）：
- 50% 仓位，首日股票涨 10% → 实际组合 +5%，旧代码报告 **0%**
- 50% 仓位，交易成本 1% → 实际 -0.5%，旧代码报告 **0%**

---

### Bug 3：回测重归一化抹掉仓位约束——"说好了 10% 上限，回测里却变成 12%"

**问题代码**（`backtest_engine.py` 旧版）：
```python
active_weights = {t: w / w_sum for t, w in active_weights.items()}
```

PositionSizer 辛辛苦苦把每只股票压到 10% 以下、留了 5% 现金。然后回测引擎一句"统统加起来重新除以总和"，把：
- 现金从 5% 抹成 0%
- 被压到 10% 的股票又被放大回 12%
- 板块 25% 上限被重新突破

**"我们修好了仓位约束"，但回测跑的根本不是约束后的组合。**

## 六、关键概念词汇表

| 概念 | 为什么重要 |
|---|---|
| **前视偏差** | 如果因子计算能看到未来行情，任何 IC 分析和选股验证都是假的 |
| **因果窗口** | 训练数据必须是决策日之前的数据，测试数据必须是之后的数据。互不重叠 |
| **开盘买入回测** | 真正的"买入后持有"应该用首日开盘价进入，不是收盘价 |
| **配对比较** | 不能说"A 的中位数 5%、B 的中位数 3%，所以 A 更好"——必须用同一天、同一窗口的差值比较 |
| **预注册** | 先定好晋级标准，再跑数据。不能在看到结果后调整阈值 |
| **开发集 vs 封存集** | 开发集用于选因子、调参数（可以看）。封存集只跑一次验证（不能看）。避免过拟合 |
| **Policy Validator** | 在 Symphony 生成计划后、执行前，强制检查"数据先于因子""覆盖不足必须停止"等规则 |
| **Agent ≠ 自动提分** | 只有实际改变持仓的 Agent 决策才影响分数。Symphony/Cron/Evolution 主要提升工程可靠性 |

| 时间 | 相信的分数 | 实际上 |
|---|---|---|
| 早上 | 81.7 | 前视偏差 + 回测 bug 污染的假分数 |

**诚实测量是一条不断下降再上升的曲线。** 先跌（修 bug → 发现更低的真相），再涨（找到真正有效的机制）。这个过程在量化投资中是完全正常的——大部分团队的"高分"经不起独立复现。我们现在的位置是：**测量工具可靠了，策略方向清楚了，样本外验证是唯一剩下的硬仗。**
