# 08 — 因子有效性验证与 IC 分析

## 怎么判断一个因子有没用

量化里有个核心指标叫 **IC（Information Coefficient，信息系数）**。

通俗理解：IC 衡量的是"因子打分高的股票，未来真的涨了吗"。

### IC 怎么算

```
1. 在每个窗口开始的时候，对所有股票计算因子 Z-score
2. 在窗口结束的时候，看每只股票的实际收益率
3. 对 Z-score 和收益率做 Spearman 秩相关

如果 IC > 0：因子分高的股票真的涨得多 → 因子有效
如果 IC ≈ 0：因子分高低和涨跌没关系 → 因子没用
如果 IC < 0：因子分高的反而跌得多 → 因子有效但方向反了
```

### 我们今天跑出来的数据

| 因子 | Mean IC | IR | 判决 |
|------|---------|-----|------|
| momentum_20 (20日动量) | +0.72 | +16.0 | 最强——涨的继续涨 |
| turnover_momentum (换手率动量) | +0.69 | +12.2 | 很强 |
| rsi (相对强弱) | +0.58 | +7.3 | 强 |
| momentum_60 (60日动量) | +0.41 | +3.1 | 有效 |
| reversal_5 (5日反转) | **-0.39** | -3.7 | 有效但**方向反了** |
| max_drawdown (最大回撤) | -0.38 | -2.1 | 有效——跌得少的真的表现好 |
| volatility (波动率) | +0.07 | +0.4 | 几乎无用——但别急着砍 |
| pe_ttm (市盈率) | -0.01 | -0.1 | 完全无用 |
| pb_mrq (市净率) | -0.07 | -0.5 | 完全无用 |
| roe (净资产收益率) | +0.08 | +0.5 | 微弱且不一致 |
| volume_trend (成交量趋势) | 0.00 | 0.0 | **死了** |

### IR 是什么意思

IR = Mean IC / Std IC，即"信号 ÷ 噪声"。

- IR > 2.0：信号很强，稳定（momentum_20 的 IR=16 是顶级的）
- IR < 1.0：信号弱，不稳定（PE 的 IR=-0.1 和抛硬币差不多）
- IR < 0.5：基本等于噪声

## 为什么基本面因子在 20 日窗口上失效

这不是"基本面因子不好"——是**时间尺度不匹配**。

### 信号的半衰期

每个因子都有一个"半衰期"——信号在多久之后开始失效：

| 因子类型 | 更新频率 | 半衰期 | 和 20 日窗口 |
|------|------|------|------|
| 动量(20日) | 每天 | ~10 个交易日 | ✅ 匹配 |
| 反转(5日) | 每天 | ~5 个交易日 | ✅ 匹配 |
| PE/PB | 每季度 | 3-5 年 | ❌ PE 在 20 天内根本没变 |
| ROE | 每季度 | 3-5 年 | ❌ 同上 |

### 一个比喻

假设你要预测一个人接下来 20 分钟会不会跑得快：

- 技术面因子 = 看他最近 5-20 分钟的速度变化（高度相关）
- PE = 看他三个月前体检报告上的体脂率（几乎无关）

不是体脂率不重要——是"预测 20 分钟后的速度"这个任务里，体脂率的信息已经被近期速度变化覆盖了。

### 为什么 IC 分析比"看起来合理"更可靠

人脑天然会为任何因子找到"合理"的解释：

- "PE 低的股票被低估了，长期会涨" ← 听起来合理
- "ROE 高的公司盈利能力强" ← 听起来合理
- "成交量放大说明有资金进场" ← 听起来合理

但 IC 数据不会讲故事——它只看因子和未来收益之间的统计关系。PE 的 IC = -0.01 意味着"PE 低的股票未来 20 天表现更好"这个命题在统计上**不成立**。

## 如何用 IC 分析指导因子选择

1. **|Mean IC| > 0.1 且 IR > 2.0** → 强因子，保留
2. **|Mean IC| > 0.05 且 IR > 0.5** → 弱因子，考虑保留或降权
3. **|Mean IC| < 0.05 或 IR < 0.5** → 噪声，砍掉或改为约束条件

### 特殊情况：风险控制因子

volatility 的 IC 是 +0.07（几乎零预测力），但这不是砍掉它的理由。

volatility 在这个策略里的角色不是**预测收益**（alpha），而是**控制风险**（risk control）。它的逻辑不是"低波动赚更多"，而是：

```
同样赚了 5%：
  A 股: 波动率 10% → 回撤小 → 比赛得分高
  B 股: 波动率 30% → 回撤大 → 比赛得分低

选A不选B不是因为A会赚更多——是因为A炸的时候炸得更少
```

回撤占比赛评分的 24/80 分。把一个防回撤的工具从模型中移除，等于自废武功。

正确做法：把 volatility 从因子打分（composite）里拿出来，变成一个**硬约束**——"波动率超过 2σ 的股票直接排除"。这样高波股票被过滤（降低回撤），但不会因为"低波动加分"而过度集中在银行股上。

## 从中学到什么

1. **不要相信"听起来合理"**：用 IC 数据验证每个因子，而不是用直觉
2. **信号时间尺度必须匹配预测窗口**：20 日窗口 → 用日频/周频因子。季频因子不适合
3. **区分 alpha 因子和风控因子**：volatility 不预测收益但要防回撤，两者不能混为一谈
4. **IC 分析不是一次性的**：每个版本都应该重新跑，因为市场状态在变
